Introduction to CSS

CSS (Cascading Style Sheet) is a sheet style language. First created in 1994 by Håkon Wium Lie and released in December 1996. This is a type of language that can be used to describe the presentation of a markup language (HTML). CSS can be used to change the color, font and layout of a web page, aswell as making the web page adaptable to different screen sizes (mobile, tablet, large screens, small screens etc.) Even though CSS is not the only way to style HTML it is the standard for styling. CSS can be used for more complex styling (animations, complex content layouts etc.) which allows for the web page to be structured efficiently.

CSS is a client-side language. This means that the browser responds to the users mouse click (when clicking a video or hovering over an image etc). This can be done as the code is already written to initiate the action.

    What is HTML?
  • HTML (HyperText Markup Language) is a language used to define the structure of web content. It is the most basic building block of the web.
  • HTML elements tell the browser how to display the content.
  • Using HTML you can define what part of the content is the 'heading', 'paragraph', 'image', 'link' and much more.
  • HTML can used other lanaguages such as Javascript which makes web pages interactive.

Types of styling

There are three different ways to style a HTML element. These are iniline styling, page-level style and external style sheets. However, before CSS was introduced, HTML had its own style tag which was used to style the elements.

    In-line styling
  • In-line styling is used within HTML tags. Works well when trying to test a new style but shouldn’t be first choice of use.
  • It has priority over page-level and external style sheets. This means if using an in-line style whilst already having a external style sheet in use, the in-line style will overide the styling from the external style sheet for that particual HTML element.
  • Pro - Easy to test a style on a specific element
  • Con - Makes the code look messy and disorganised. Can make it harder to read the code yourself and for other developers.
    Page-level style
  • Defined at the head of the html file and is a clean version of local style (in-line style) as everything is located in one place
  • Pro - Easy to manage as there is no switching between files
  • Con - Increases the HTML file size which can make the web page run slower
    External style sheets
  • Located in CSS files .css and linked to HTML file .html using the link tag in the head of the html file.
  • Best to use when styling multiple pages as one css file can be used on multiple pages
  • Pro - Makes code look clean and organised, Makes code reusable
  • Con - Not practical for small style defintions, More files to keep track of

Color

Color is an important css style. It is used to help users navigate through the web page easily. It also makes webpage look appealing to users. One of the aspects that attracts a user on the webpage is color. Color can convey different emotions, speak to different target audiences, and communicate action.

Below shows how to link a css stylesheet to a html file using the <link> element:

<link rel="stylesheet" href="style.css">

Below shows how to add color to the body html element:

body {
   color: red;
}

This is a very easy way to add color to an element, by just specifying the name of the color. However, this only works for the generic colors.

Hex Color Codes

If you want to use a color like Fawn, as it is not one of the generic colors, it will not be dislpayed on the webpage. Therefore, a hex code is used. A hex code is a 6 symbol code accompanied by the # key. Every color has a hex code which is used for styling in css. Where rr (red), gg (green) and bb (blue) are hexadecimal values between 00 and ff (same as decimal 0-255). For example, #ff0000 is displayed as red, because red is set to its highest value (ff) and the others are set to the lowest value (00). To display black, set all values to 00, like this: #000000. To display white, set all values to ff, like this: #ffffff.

body {
   color: #ffffff;   <!-- this hex code is for the color black -->
   background-color: #e5aa70   <!-- this hex code is for the color fawn -->
}

RGB & RGBA

RGB stands for red, green, blue. Each parameter (red, green, and blue) defines the intensity of the color between 0 and 255. To display black, set all color parameters to 0, like this: rgb(0, 0, 0). To display white, set all color parameters to 255, like this: rgb(255, 255, 255).

RGBA is an extension of RGB color values with an alpha channel - which specifies the opacity for a color. The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all).

Both RGB and RGBA are not as commonly used in css for choosing a color but it is a great option to have.

body {
   color: RGB(73, 31, 28);   <!-- this rgb value if for the color Deep chestnut -->
   background-color: RGBA(73, 31, 28, 0.6);   <!-- adding 0.6 (alpha value) means the color will be partially transparent -->
}

HSL

HSL stands for Hue, Saturation and Light. It is a more recent way to work with colors. Unlike hex or RGBA where to use different numbers to define a color, in HSL, Hue can be used to define the colour whilst the second and third parameters are used for the saturation and light levels which are measured in percentages 0 to 100%.

Hue ranges from 0 to 360 on the color wheel. Saturation is the amount of grey in a color, 100% being fully saturated (no grey). Lightness is the amount of white or black in a color. 0% is black, 100% is white and 50% is the normal color.

body {
   color: hsl(240, 100%, 50&);   <!-- this hsl value if for the color blue-->
}

Text

There are several different properties for formatting text. The very basic one which we have learnt in the previous section is color. By adding the color property, you can change the color of the text within that specific element.

You can align text by using the text-align: property. This allows for horizontal alignment of a text either, left, right or center of the page. You can also set it to justify, which means that each line will be stretched so it has an equal width. Alongside this, you have an option to text-align-last:. This is to set the last line of text and takes the same values as text-align.

Another property is text-decoration: . This can be used to set or remove decorations from text. The values for this property are underline, overline, line-through or none.

text-transform: property allows you to specifiy if you want the text to be uppercase, lowercase or if you want the first letter of each word to be capatilize.

There are several different properties which allows for text-spacing. These are text-indent: - used to specify the indentation of the first lineof text. letter-spacing: - space between each letter, word-spacing: - space between each letter, line-height: - space between the lines.

A final text property is text-shadow:. This property adds shadow to the text by specifying the horizontal and vertical shadow (px). You can add color to the text shadow by specifying the color after the horizontal and vertical shadow. Moreover, you can specify a blur effect (px) to the shadow which goes before the color and after the vertical shadow value.

p {
   text-align: center;   
   text-decoration: underline;
   text-transform: uppercase;
   word-spacing: 5px;
   text-shadow: 1px 2px blue;
}
All the above values have been applied to this text.

Font

Choosing the right font for your website is vital. It makes a huge impact on the readers experience on the website. It is crucial to use a font that is easy to read, has the correct color and size as all these factors could be a drawback to readers if not used appropriately.

You can choose the type of font you want your text to be in by using font-family:. There are 5 main font-families which can be used in CSS and these are serif, sans-serif, monospace, cursive and fantasy. All the font names belong to one of these generic font-families. When choosing a font, it is best to put your main font in " " and have a fallback font (generic family) which is not within quotation marks. Doing this allows for maximum compatibility between browsers. Also as not every font can be found by the web so having a fallback font ensures that your text has a font which can be used and shown on the browser.

Font styling is another way to make your text appeal to the readers. The font-style: property takes three values normal - text shown normally, oblique - text is leaning and italic - text is in italics.

You can make your font bold by using the font-weight: property. This property takes values normal, bold, bolder, lighter as well as numbers 100 to 900 (in hundreds). 700 is bold and 400 is normal.

A very important font property is the font-size:. Being able to define the size of the font is important for web designing. This property takes three different values which all give a similar outcome. You can use px, em or %. 1em is equal to the current font-size. 16px is the default text size in browsers. You can also use vw which stands for viewport width. This means that the font will change according to the size of the viewport. 10vw = 10% of viewport width.

p {
   font-family: "Papyrus", serif;   
   font-style: italics;
   font-weight: bold;
   font-size: 1.2vw;
}
All the above values have been applied to this text.

Backgrounds

The background: property in css is used to add background effects to the elements. To change the color of the background, you use the background-color: property and use any color (refer to color section) as the value. You can do this to any html element you want.

Similar to adding a background color, you can add a background-image:. This can be done by adding the url of the image you would like to use. However, by default the image is set to repeat and therefore, for this not to happen you can use the background-repeat: property and set the value to no-repeat.

By setting the background-attachment: you can specificy if you want the background to scroll as you scroll through the page or stay fixed (wont scroll with rest of the page). You can also specifiy the position you want the background image to be on the page by using the background-position: property.

Using a shorthand property can make it easier to specify the values you want to give to each background property. This is done by using background: and then specifying each value in this order:

  1. background-color:
  2. background-image:
  3. background-repeat:
  4. background-attachment:
  5. background-position:

body {
   background: blue url(" ") no-repeat scroll right top;
}

Borders

The border properties are used to specifiy the style, width and color of an elements border. border-style: can be used to choose what type of styling you want your border to have. There are several options to choose from such as:

  • dotted
  • dashed
  • solid
  • double
  • groove
  • hidden

The border width property is used to define the width of the four borders. The width can be set using different units (px, em, cm) or three pre-defined values: thick, medium, thin. Can add color to the border by using the border-color: property and setting the color by name, hex, rgb or hsl.

To make it easier and your css code shorter, you can use the shorthand property for this. This can be done by using border: and setting the values in this order:

  1. border-width
  2. border-style
  3. border-color

To make the border more rounded, you can use the border-radius: property and again use value in px.

body {
   border: 5px double green;
   border-radius: 6px;
}

All the above values have been applied here.

Display

The display: property specifies how an element is dislpayed. Every HTML element has a display which is either in-line of block.

An in-line element takes up as much width as necessary and does not start on a new line. Examples of in-line elements are:

  • <span>
  • < a >
  • <img>

A block element always starts on a new line and takes the full width available. Examples of block elements are:

  • < div >
  • <h1> - <h6>
  • <p>
  • < form >
  • <header>
  • < footer >
  • <section>

Even though every element has a default display, this can be overriden by changing the display value

Lists

In HTML there are 2 types of list, ordered and unordered. The CSS list property allows you to set different items for both list types, use images as markers or add a background color. You can specify the list marker by using list-style-type and setting it to values such as square, circle, upper-roman or lower-alpha. You can use an image and set it as the marker - list-style-image:.

By using the list-style-position: property you can set whether you want the bullet points to be on the outside or inside.

ul {
   list-style-type: sqaure;
   list-style-position: inside;
   list-style-image: url(" ");
}

The list shorthand property is best to use when trying to declare all the aboce properties for a list. The order of the values are:

  1. list-style-type
  2. list-style-position
  3. list-style-image

Id's & Classes

Id's and classes allow you to define predeined chunks of css styling and then use them on HTML elements. These are known as simple CSS selectors.

ID's

ID = identifier

The id element is unique and so can only be used to select one element. To use an id the # should be used followed by a name (chosen by yourself - best to be related to the type of styling you will assign the id). This can then be used within a HTML element and the styling will be applied. Id's are mainly used for javascript or elements that only appear once (header, footer, main).

#color {
   color: red;
   background-color: black;
}

<body id="color" > </body>

Classes

The class element is simiar to the id element in terms of its use for styling html elements. unique and so can only be used to select one element. However, one class can be used for several html elements. To use a class in css, you write a period/fullstop (.) followed by class name (should be related to the styling, as will be easy to remember). You can also use mutiple classes for one element by just specifying the class name within the html elemnt.

.color {
   color: red;
   background-color: black;
}
.text {
   align-text: justify;
   text-transform: uppercase;
}

<p class="color" > </p>
<section class="color text" > </section>

Flexbox

Flexbox layout allows for a flexible responsive layout structure without having to use float or positioning. Before the flexbox layout was introduced, the four layout modes that were used were, inline, block, table (two-dimensional table layout) and positioned (explicit layout of an element).

Flex-container

First the flex container needs to be defined - .container. Then the element should have display: flex. This turns the container into a flex container. By doing this, it becomes possible to align any children of that element into rows or columns. You can do this by adding a flex-direction: property and setting the value to row, column, row-reverse or column-reverse. The row value stacks the flex items horizontally (left to right) whilst row-reverse stacks them from (right to left). The column value stacks the flex items vertically (top to bottom) whilst column-reverse stacks the items (bottom to top).

There are several options on how to align items along the line which is the main axis. Using justify-content:.

  • center - align flex items to the center of the container
  • flex-start - align items to the start of the flex container (this is the default alignment)
  • flex-end - align items at the end of the flex container
  • space-between - the first and last item will be pushed to the edge of the container and others items will have space distributed between them
  • space-around - space around every item within the flex container. The space will not be even
  • space-evenly - Even space around all flex items

.flex-container {
   display: flex;
   flex-direction: row;
   justify-content: center;

The align-items: property is similar to justify-content. Align-items aligns the items along the cross axis. For rows this is vertical and for columns it is vertical. Some of the values for align-items are similar to that of justofy-content:

  • flex-start - align items are start of container. Row = top of container / Column = left of container
  • flex-end - align items at end of container. Row = bottom of container / Column = right of container.
  • center - align items in center. Row = vertically / Column = horizontally
  • stretch - stretch items to fill container (is a default value)
  • baseline - aligns items to their baseline. (is a text concept - the lines the letters sit on)

The align-content: property can be used to align the flex lines. There are many values that this property takes:

  • center - lines are packed towards the center of the container
  • flex-start - lines are packed to the start of the container
  • flex-end - lines are packed to the end of the container
  • space-between - first and last line pushed to edge of container whilst other items are distributed within the container
  • space-around - lines distributed within the container, with space at the each end
  • space-evenly - lines evenly distributed within the container
  • stretch - default value

Have the flex-wrap: property. This property is used to split the flexitems into mulitple rows or columns. By default a flex container will fit all the flex items together. The values for flex-wrap are:

  • wrap - wrap items into multiple lines from top to bottom (rows) or left to right(columns)
  • no-wrap - default setting
  • wrap-reverse - wrap items but in the opposite direction

Flex-items

The flex-shrink: property is a property used on the flex items rather than the container. This allows the items to shrink if the container to too small for all the items. This property takes numbers as values. The higher the number the more it will shrink.

The flex-grow: property is opposite to flex-shrink. It specifies how much a flex-item will grow relative to the other flex items.

The flex-basis: property is used to specify the inital size of an item before making adjustments with flex-shrink or flex-grow. The units used for flex-basis are: px, em or %.

You can use the flex shorthand property flex: to set the values for flex-grow, flex-shrink and flex-basis (in this order).

An order: property can be used to specify which order you want the flex-items to appear in the container. By default the items will appear in the same order as the HTML source.

A final property which can be used on the flex items is align-self:. This property allows for aligning of individual flex items. It will override align-items property. Takes the same values as align-items.

Animations

Can use CSS to create animations of HTML elements. An animation allows for a gradual change of an element from one style to another. To use CSS animation, a keyframe animation property must first be defined.

@keyframes rule does not change the appearance of the element but is used to describe each step of change for the animation. To start the @keyframes rule should be given a name e.g @keyframes change-color. Within this you can now specifiy which color you want the element to change to. E.g - From red, To purple.

@keyframes change-color {
    from {background-color: red;}
    to {background-color: purple;}
}

To bind the keyframes to the element (either use element or class), you use the animation-name: property. The name is the same as the name you gave the keyframes rule. This tells the element that there is an animation to be used. You can use the animation-duration: property to specify how long the animation should take to complete. Without this property, the animation will not happen. Other properties include:

  • animation-iteration-count: - How many times you want the animation to occur. Can set this to "infinite" and the animation will carry on playing.
  • animation-delay: - How long of a delay before animation starts
  • animation-direction: - This can be normal, reverse(animation played backwards), alternate(animation played forward first and then backwards), alternate-reverse(animation played backwards first and then forward).

@keyframes change-color {
    from {background-color: red;}
    to {background-color: purple;}
}
.box-animation{
    animation-name: change-color;
    animation-duration: 5s;
    animation-iteration-count: 2;
    animation-delay: 10s;
}

Shown above, we used keywords "from" and "to" to define how our animation will change. We can also use percent (%) and add as many style as we want with this. Alongside this we can also change the position of the animation at each percent completion.

@keyframes change-color-position {
   0% {background-color: #D0F0CO; left: 0px; top: 0px;}
   25% {background-color: #90EE90; left: 200px; top: 0px;}
   50% {background-color: #32CD32; left: 200px; top: 200px;}
   75% {background-color: #138808; left: 0px; top: 200px;}
   100% {background-color: #004225; left: 0px; top: 0px;}
}
.box-animation{
   animation-name: change-color;
   animation-duration: 5s;
   animation-iteration-count: infinite;
}
